# Readme

## Device

* Bus 002 Device 003 ID 16c0:27db

## Deps

* [libusb](http://dcuddeback.github.io/libusb-rs/libusb/)

## Behavior

* Reads card number as keybaord input including a new line, e.g.:
  * `0013110988` <= with new line
